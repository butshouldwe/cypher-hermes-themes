# Code of Conduct

Be kind, be constructive, assume good faith. This is a small project about
making a tool pleasant to use — bring that energy.

- Critique themes, not people.
- No harassment, discrimination, or bad-faith noise.
- Maintainers may edit, hide, or remove contributions that don't fit, and may
  block accounts that repeatedly break these terms.

Report issues to **conduct@butshouldwe.com**. We adapt the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/) — the short
version is: don't be a jerk, and help the room stay welcoming.
