# Security Policy

These themes are declarative YAML + scoped CSS for the Hermes dashboard — no
code executes from them. The realistic surface is a CSS selector that
unintentionally hides or miscolors part of the UI (a legibility bug, not a
vulnerability).

If you spot something that could **hide a control, obscure a warning/error
state, or leak visual information** across the dashboard, please report it
privately rather than opening a public issue:

- Use [GitHub private vulnerability reporting](https://github.com/butshouldwe/cypher-hermes-themes/security/advisories/new), or
- email **security@butshouldwe.com**

We'll acknowledge within a few days. Thank you for keeping the room safe to
read.
